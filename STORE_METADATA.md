## Description
Object Measurer turns your phone into a precision measurement tool. Simply point your camera at any object, and get instant measurements in millimeters, centimeters, or inches.

**How it works:**

📐 **Calibrate** — Choose a reference object (credit card, coin, or custom) to establish scale.

📷 **Photo Mode** — Capture a photo and get precise server-side measurements using advanced computer vision (background removal, edge detection, and distance calculation).

⚡ **Real-Time Mode** — See measurements live on your camera feed with on-device processing for instant results.

📊 **History** — All your measurements are saved so you can review them anytime.

**Key Features:**
• Measure width, height, and area of any object
• Choose between real-time and photo measurement modes
• Multiple reference object presets (credit card, dollar bill, A4 paper)
• Custom calibration with any reference object
• Unit switching: mm, cm, or inches
• Dark mode interface
• Measurement history with full dimensions
• Secure cloud sync across devices

**Technology:**
Object Measurer combines on-device and server-side computer vision for the best balance of speed and accuracy. On-device processing gives you real-time feedback, while server-side analysis provides precision measurements within ±2mm.
